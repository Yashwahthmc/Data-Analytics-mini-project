# 🌍 Global Trade Analytics System

## 📌 Project Overview

The **Global Trade Analytics System** is a data analytics project that studies international trade patterns between countries using real-world export data. The system analyzes trade relationships, identifies key export destinations, and visualizes trends over time.

This project is developed as part of the **Artificial Intelligence Laboratory (Mini Project)** under the **AI & ML program (Anna University)**.

---

## 🎯 Objectives

* Analyze global trade data between countries
* Identify top exporting countries
* Study year-wise trade trends
* Perform commodity-wise analysis
* Visualize trade insights using graphs

---

## 📊 Dataset

The dataset is obtained from **Kaggle** and contains export trade data from **2010 to 2021**.

### Dataset Features:

* `HSCode` – Product classification code
* `Commodity` – Product description
* `value` – Export value
* `country` – Destination country
* `year` – Year of export

## 📈 Key Results

* 🇺🇸 USA is the top export destination
* 🇦🇪 UAE and 🇨🇳 China are major trade partners
* Export trends show overall growth from 2010 to 2021
* Major export categories include:

  * Mineral fuels
  * Precious stones
  * Machinery
  * Vehicles
